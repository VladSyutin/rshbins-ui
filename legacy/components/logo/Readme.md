## Legacy Logo

React 17-compatible brand logo for FOS integration. SVG paths are inlined directly.

### Files

- `Logo.jsx` — component implementation
- `styles.css` — logo styles
- `index.js` — default export entry point
- `index.d.ts` — TypeScript type declarations

### Example

```jsx
import React from 'react';
import Logo from './index';

export default function Example() {
  return <Logo />;
}
```
